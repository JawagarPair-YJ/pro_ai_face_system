
# AI FACE RECOGNITION ATTENDANCE SYSTEM

## INSTALLATION

1. Install Python

2. Open project in VS Code

3. Create virtual environment

python -m venv venv

4. Activate environment

venv\Scripts\activate

5. Install packages

pip install -r requirements.txt

6. Run project

python app.py

7. Open browser

http://127.0.0.1:5000

Press Q to close webcam.
